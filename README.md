# OwO-Style Discord Bot

Features:
- Economy system
- Daily rewards
- Balance command
- Hunt command
- SQLite database
- Ready for Railway deployment

## Setup

1. Create a Discord bot application.
2. Copy `.env.example` to `.env`
3. Add your token.
4. Install dependencies:

```bash
npm install
npm start
```

## Railway

Deploy directly from GitHub and set `DISCORD_TOKEN` in Railway variables.
